<오늘의 목표> - UI
터렛을 구매하는 상점을 만들어 보자

<오늘의 과제>
1. 스크린 하단에 터렛 구매 버튼을 2개 만든다

2. 스크린 하단의 첫번째 터렛 구매 버튼 클릭 후에만
  맵에 터렛을 설치할 수 있다
  Debug.Log("기본 터렛을 선택 하였습니다!!"); 
1) 첫번째 구매버튼 클릭 :  설치할 터렛에 기본 터렛(프리팹)을 저장
2) 타일을 클릭하면 저장된 터렛을 설치하세요
3) 만약 저장된 터렛이 없으면(선택버튼을 누르지 않으면) 설치 실패 처리하세요
  (구매버튼을 누르지 않으면 터렛 설치를 실패해야 합니다.)
  구매버튼을 누르지 않고 타일을 선택하여 터렛 설치하려고 하면 실패처리
  Debug.Log("터렛을 설치하지 못했습니다.!!");

3. 타일 위에 버튼(UI)이 있을 경우 타일 선택 또는 터렛 설치가 안되게 하기

4. 기본 터렛의 프리팹 복사 기본 터렛 프리팹2 만들고 두번째 버튼 연결해서 
 선택(구매) 기능, 설치 기능 구현
=  두번째 구매버튼 클릭 : Debug.Log("다른 터렛 선택!!");
=  두번째 구매버튼 클릭시 : AnotherTurretPrefab 저장
= 맵타일 클릭하면 AnotherTurret 맵에 설치

<필요 스킬>
1. UI - Panel (Layout Group) , Button
2. EventSystem.current.IsPointerOverGameObject()
